Summary

HTML is a HyperText Markup language which is based on text with some basic text with some computer commands markup to tell it how to behave.

This is one of the pieces that consolidates the internet. Html is the protocol between the user computer and servers. Servers are computers meant to host most of the internet's data.

--

The syntax

An element is composed of opening and closing tags. Opening tags are < > and it's the way to tell computer to do something from this point forward.

Closing tags are </> needed to tell it to stop that element from taking effect. This changes depending of the type of elements. There are void elements that don't require a closing tag.

---

Attributes

For now two types, href for adding links to text, and src to define the location of a image file.
